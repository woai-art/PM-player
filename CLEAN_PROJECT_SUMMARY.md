## 🚀 **Простой запуск**

### **Один файл запуска**
```bash
start-simple.bat
```

### **Или вручную**
```bash
node simple-server.js
```

### **Использование**
1. Откройте http://localhost:3001
2. Вставьте URL видео
3. **Load** → **Play** или **Download**
